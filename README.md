## 🦄 Instalación 🦄

  * Primero crea y entra a la carpeta que contrendrá el proyecto
  * Abrir la terminal 
    - git clone https://github.com/grupo-desarollo-web-fullstack/yapuy-project.git
    - cd yapuy-project
    - Levantar un servidor de estáticos y correr la aplicación
